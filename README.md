# Google Play Store Analytics Dashboard

## Project Overview
This project transforms Exploratory Data Analysis (EDA) findings into an interactive Power BI dashboard.

## Tools Used
- Power BI
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dashboard Pages
1. Executive Overview
2. App Performance Analysis
3. User Sentiment Analysis

## Key Insights
- Family category contains the highest number of apps.
- Most apps are free.
- Average app rating is 4.19.
- Positive reviews account for over 60% of user sentiment.
- Social category receives the highest review volume.
